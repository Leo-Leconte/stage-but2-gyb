# stage-but2-gyb

Bienvenue sur le projet **stage-but2-gyb** !

Nous réalisons une application web accessible a tous les collaborateurs, permettant de visualiser qui est en stage (ou apprentissage), quand, dans quel service, et de centraliser les documents lies a chaque stage (ou contrat d'apprentissage).

1.  [Initialisation du Projet](#-initialisation-du-projet)
2.  [Suivi du travail](#-suivi-du-travail)

---

## Initialisation du Projet

-   `cd front`
-   `npm create vite@latest front -- --template react-ts`
-   `cd ../back`
-   `npm init -y`
-   `npm install`
-   `npm install react-router@6 react-router-dom@6`
-   `cd ..`
-   `npm install --save-dev @testing-library/react @testing-library/dom @types/react @types/react-dom` (pour les tests react)
